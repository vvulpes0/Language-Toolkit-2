Documentation

Installation instructions are available on the home page. User guides for the pleb format and its interpreter plebby, as well as the factorize program, are available as a gzipped tarball. Alternatively, you can checkout the lastest development snapshot via SVN:

svn checkout https://lutras-hacking.ddns.net/svn/Language-Toolkit-2/branches/docs

This SVN documentation will apply to the latest development snapshot of the code.

If you wish to use the Local Language Toolkit as a library, much of the code has associated Haddock documentation.
