Installation

  1. Install GHC, the Haskell parallel libraries, GraphViz, and ImageMagick:
    sudo apt-get install ghc libghc-parallel-dev graphviz imagemagick
  2. cd to the directory in which you will run the toolkit.
  3. Download the current version of the toolkit.
  4. Unpack the tarball:
    tar xzf Language-Toolkit-2-<versionNumber>.tar.gz
    Where <versionNumber> will be something like 0.01a.r107. This will create a directory named <versionNumber>.
  5. cd <versionNumber>.
  6. Download the documentation and unpack it:
    tar xzf <docs.tar.gz>
    This will create a directory docs in your current directory.
  7. Download the example pleb and plebby files and unpack them:
    tar xzf examples.tar.gz
    This will create a directory plebFiles in your current directory.
  8. Compile plebby
    ghc -O2 -threaded -with-rtsopts='-N -C0.001 -qg1' plebby
  9. Optionally download the StressTyp2 archive.
  10. Unpack the archive:
    tar xzf st2-v1-archive-0415.tar.gz
    This will create the directory st2-v1-archive-0415 in your current directory.
  11. Run plebby and test it
    ./plebby
    > :import <plebFiles>/yidin.plebby
    > :display yidin
    Where <plebFiles> is the path to the directory with the example files. This should display the automaton for Yidin, as described in the StressTyp2 database.
    > :readJeff st2-v1-archive-0415/transducers/fsasJeff/219_Yidin_SPD.fsa
    > :equal it yidin
    If all has gone well, this should return True. You may :quit when you are done.
  12. If you would like to use factorize, first compile it:
    ghc -O2 -threaded -with-rtsopts='-N -C0.001 -qg1' factorize
  13. Next download the precompiled non-strict constraints.
  14. Unpack them
    tar xzf compiled.tar.gz
    This will create a directory Compiled along with the compiled constraints and the file Compiled/constraints, which are required for factorize.
  15. Run factorize
    ./factorize st2-v1-archive-0415/transducers/fsasJeff/*
    (You may wish to choose a subset of the lects to start with, but the whole library should take less than five minutes.) The results will be in the directory Results.
